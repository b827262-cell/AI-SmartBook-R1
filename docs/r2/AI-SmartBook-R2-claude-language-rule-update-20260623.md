## 變更目的

使用者要求調整 Claude Code 的語言輸出規則：

- 編程進行過程中改用**英文**描述（進度更新、工具呼叫說明、中間步驟）
- 僅在每個任務**完成時的結案報告**才切換回繁體中文

| 檔案 | 類型 |
|------|------|
| `~/.claude/CLAUDE.md` | 全域 Claude Code 設定（所有專案共用）|

```markdown
## 語言規則（強制）
- 所有回應、說明、進度敘述、思考摘要一律使用「繁體中文」。

- 程式碼、變數名、檔案路徑、shell 指令、API 名稱保持英文原樣，不要翻譯。
```

---

## 修改後

```markdown
## 語言規則（強制）
- 編程過程中所有進度敘述、工具呼叫說明、中間步驟更新一律使用「英文」。
- 僅在任務完成時的最終結案報告（termination report）使用「繁體中文」。
- 嚴禁使用日文（日本語）及簡體中文。
- 程式碼、變數名、檔案路徑、shell 指令、API 名稱保持英文原樣，不要翻譯。
```

---

## 規則說明

| 情境 | 語言 |
|------|------|
| 進度敘述（工具呼叫前後的說明） | 英文 |
| 工具呼叫 description | 英文 |
| 中間步驟更新、調查結果 | 英文 |
| 任務完成的最終結案報告 | 繁體中文 |
| 程式碼、變數名、路徑、指令 | 英文（保持原樣）|
| 日文、簡體中文 | 嚴禁使用 |

適用於此帳號下所有 Claude Code 工作階段與專案。
